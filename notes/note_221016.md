### 2. Framework vs. Library



|        | Framework                  | Library                        |
| ------ | -------------------------- | ------------------------------ |
| 설명   | 기반 구조                  | 편의 도구                      |
| 주도권 | 프레임워크가 주도권을 가짐 | 사용자(개발자)가 주도권을 가짐 |
| 자유도 | 상대적으로 작음            | 상대적으로 큼                  |
